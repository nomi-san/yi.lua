# yi.lua
Yi - C interface library for Lua

### Features
- [x] Multi-threading
- [x] Enumerate
- [x] Data structure
- [x] Calling C function
- [x] Callback function / closure
- [ ] Classes

### Platform support

**Compiler** | **Arch**
:--|:--
Win32-MSVC | `x86 / ia32` `x64`
Win32-GCC | `x86 / ia32 / i686`
Win32-TCC | `x86 / ia32`
Linux-GCC | `x86 / i686`
